Sistema de Gerenciamento Ninja

Funcionalidades:
- Cadastro de Ninjas
- Cadastro de Missões
- Vinculação Ninja x Missão
- Validação de participação por rank
- Totalizadores
- Views SQL

Tecnologias:
- Java Swing
- MySQL
- JDBC

Como executar:
1. Execute o arquivo database.sql
2. Configure usuário e senha do MySQL em ConnectionFactory
3. Execute MenuPrincipalWindow
